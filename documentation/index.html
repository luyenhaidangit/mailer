Introduction

Mailer – Email Marketing Application is the easiest way to send email marketing newsletters! Create your brands, add campaigns and monitor progression. Mailer has powerful features to create dynamic lists using different segments. Stop wasting monthly subscription fees and make your newsletter marketing more professional.

Mailer is developed using Laravel 8.1, Vue.js & MySQL database. It’s easy to install and you’ll get free updates forever.

Why Mailer?

  • Dashboard

    See all statuses of the application from dashboard and stay up to date.

  • Campaign

    Set different types of campaigns and send emails automatically.

  • Email

    Easily explore all emails for each campaign with a detailed view.

  • List

    Create and manage subscribers as your audience within a list .

  • Segments

    Target the right people using behavior logic.

  • Subscriber

    Import subscribers from existing system and start campaign.

  • Templates

    Create and customize beautiful & responsive email templates as you want.

  • Users & Roles

    Manage your uses with different roles and permissions.

  • Custom Field

    Store more specific subscriber information using custom fields.
