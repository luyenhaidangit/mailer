Quick start guide

For user accessibility and better maintenance, the full application is separated into two level:
  1. App level
  2. Brand level

App Level

  • After login into your main admin panel please configure the Application settings as well as Brand settings, and make sure you set up all the Important settings in order to run the full application.

  • You can manage the user and their role of this app in Users & Roles.

  • You can create and manage Brands and Brand Group from the app level or just start with the default one.

Brand Level

This is the place where all the magic happens.

  • Go to your Brands Click on a brand, It will open that brand's dashboard.

  • Go to your Subscribers Create or Import Subscribers.

  • Go to Segments Create Segments

  • Go to Lists Create a new list with segments and subscribers.

  • Go to Templates Create a new template.

  • Go to Settings and set necessary settings.

    S: N: For delivery setting if you want to use the default one, then please make sure the brand delivery setting is saved from the app level.

  • Go to Campaign Create a campaign step by step.

And you are ready to go.
